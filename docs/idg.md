# Infrastructures de Données Géographiques (IDG)

## Eléments de définition
### 1. IDG généraliste ou territoriale (diffusion multi-thématique)
### 2. IDG thématique (se concentre sur un enjeu territorial)
  - risques : *géorisques*
  - urbanisme : *géoportail de l'urbanisme*


### 3. Evolution sémantique
**IDG** vers **plateforme de données géographiques** vers **plateforme territoriale de données** :
- Le terme IDG focalise plutôt le volet technique / technologique.

## Services
- recherche
- consultation
- téléchargement
- transformation
- appels (WMS, WFS, CSW -- moissonage de catalogue)

## Stratégie européenne de la donnée (février 2020)
Espaces Communs de Données (**ECD**) ou *Data Space* :
- Un espace de données réunit les infrastructures de données pertinentes et les cadres de gouvernance afin de faciliter la mise en commun et le partage des données.
